TKX is a lightweight SQL injection scanner designed for educational purposes, penetration testing practice, and security awareness training.

## üöÄ Features

- **Multiple Detection Techniques**
  - Error-based SQL injection detection
  - Time-based blind SQL injection  
  - Union-based SQL injection detection
  - Boolean-based blind SQL detection

- **Advanced Capabilities**
  - WAF bypass techniques
  - Database fingerprinting
  - False positive filtering
  - Custom payload management

- **User-Friendly Interface**
  - Color-coded CLI output
  - Multiple output formats (JSON, Text)
  - Progress tracking
  - Easy installation (single dependency)
